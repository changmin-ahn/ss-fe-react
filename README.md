## 조건부 렌더링

컴포넌트 외부에서 전달된 데이터에 따라 조건부로 마크업을 렌더링하는 방법을 학습합니다.

- if 또는 switch 문 활용
- 제어 흐름 연산자(`&&`, `||`) 활용
- 터너리(ternary, 삼항 연산자 식) 활용
- nullish 병합 연산자(`??`) 활용
- 옵셔널 체이닝(`?.`) 활용
- 표시(show) & 감춤(hide)
- 스크린 리더(screen reader) 지원
